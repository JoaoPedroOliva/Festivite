# Festivite

Projeto web estático para o site do Festivite. Contém HTML, CSS e assets organizados por pastas.

## Estrutura

```
assets/
  icons/
  images/
scripts/
styles/
  fields/
index.html
```

## Pré-requisitos

- Navegador moderno (Chrome, Edge, Firefox ou Safari).

## Como executar

Opção rápida:

1. Abra o arquivo `index.html` no navegador.

Opção com servidor local (recomendado para caminhos relativos):

## Estilos

Os estilos globais e de layout estão em `styles/` e os componentes em `styles/fields/`.

## Assets

Imagens e ícones ficam em `assets/images/` e `assets/icons/`.
